Deal Field Mapping

Map your deal data fields to Pipedrive deal fields.

Available Source Fields
  • NPO_ProjNr - Project Number
  • NPO_ProjName - Project Name
  • NPO_KSumme - Initial Amount
  • NPO_ASumme - Final Amount
  • NPO_KDatum - Start Date
  • NPO_ADatum - End Date
  • Status - Deal Status
